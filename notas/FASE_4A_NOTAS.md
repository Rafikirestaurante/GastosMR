# Fase 4A — Navegación unificada, Inicio y Pendientes

Base utilizada: `1.6.5_control-gastos-milena-fase-3f-configuracion-persistente.zip`.

Versión generada: `1.7.0`.

## Cambios

1. Navegación unificada en computador y celular con las secciones Dashboard, Inicio, Rafa, Pendientes y Config.
2. En móvil se reemplazó la barra inferior por un menú desplegable mediante botón de tres líneas.
3. Nuevo registro e historial se integraron en la sección Inicio.
4. Inicio ofrece vista en tabla o tarjetas y muestra saldo acumulado en ambas vistas.
5. Se creó la sección Pendientes, ordenada desde la tarea más próxima hasta las tareas sin fecha.
6. Desde Pendientes se puede editar texto, fecha, hora y recurrencia, completar o borrar tareas.
7. Las tareas completadas se muestran en un bloque separado.
8. El chatbot se simplificó para crear recordatorios solamente; no muestra listados ni acciones de administración.
9. Se conserva el backend `1.6.5-fase-3f-configuracion-persistente`, la configuración persistente y la cola local-first.
10. Se corrigió el README para eliminar instrucciones antiguas que pedían editar el ID y el token directamente en `Code.gs`.
